# Projet 3 — Audit et sécurisation d’un système Linux

## Objectif
Analyser et sécuriser un système Ubuntu afin de comprendre les bases de la cybersécurité.

## Étapes
- Mise à jour du système
- Configuration du firewall (UFW)
- Installation de Fail2Ban
- Analyse réseau avec Nmap

## Résultat
- Un seul port ouvert en local
- Système sécurisé
