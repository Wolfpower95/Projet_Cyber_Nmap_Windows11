# Projet Cybersécurité – Audit Nmap Windows 11
Ce projet montre un audit réseau d’une machine Windows 11 virtuelle avec Nmap.

## Objectif
- Identifier les ports ouverts et les services
- Analyser la sécurité (SMB, TCP)

## Résultats
- Ports ouverts : 135 (RPC), 139 (NetBIOS), 445 (SMB)
- SMB sécurisé (message signing activé)

## Conclusion
La machine est correctement configurée et sécurisée. 
Recommandations : maintenir le système à jour, activer le firewall, désactiver les services inutiles.
