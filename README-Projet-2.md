# Projet 2 - Audit de sécurité Windows

## Objectif
Ce projet consiste à réaliser un audit de sécurité d'une machine Windows 11 virtuelle.

## Étapes réalisées
- Analyse des services Windows
- Analyse des comptes utilisateurs
- Vérification du pare-feu

## Outils utilisés
- Windows 11
- Invite de commandes (CMD)
- Machine virtuelle

## Résultat
L'analyse montre que la machine possède un pare-feu actif, des comptes utilisateurs contrôlés et des services configurés correctement.

## Compétences acquises
- Audit sécurité Windows
- Analyse des services
- Gestion des comptes utilisateurs
- Vérification du pare-feu
